# MISP-Threat-Intelligence Investigation
## Decription
SOC threat intelligence investigation using MISP to analyze ransomware activity, DDoS malware, threat actors, and vulnerability exploitation with MITRE ATT&amp;CK mapping.

## OVERVIEW
This project demonstrates hands-on threat intelligence analysis using the MISP platform.
The investigation focused on ransomware activity, threat actors (including Turla Group), DDoS malware campaigns, and vulnerabilities affecting enterprise communication systems.
The goal is to simulate real SOC threat intelligence work by analyzing indicators of compromise (IOCs), threat actor behavior, and security reports.

## OBJECTIVES
In this investigation, I worked on:
- Analyzing ransomware-related threat events
- Researching known threat groups (e.g., Turla)
- Identifying malware indicators and artifacts
- Studying DDoS botnet behavior
- Reviewing CVE exploitation in enterprise systems
- Extracting intelligence from MISP events and threat feeds

## TOOLS USED
- MISP Threat Intelligence Platform
- Web-based threat intelligence research (inside lab environment)
- MITRE ATT&CK framework
- Threat actor and malware datasets
